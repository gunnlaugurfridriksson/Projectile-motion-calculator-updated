# Projectile Motion Calculator

A Python projectile motion calculator that uses the work-energy theorem to estimate the initial speed of an object from applied force, distance, and mass.

The program calculates:

- Initial speed
- Horizontal velocity
- Vertical velocity
- Time in air
- Maximum height
- Range
- Projectile trajectory graph
